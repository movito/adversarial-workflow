## Key Changes

### Bug Fixes
- **Environment loading**: Use explicit `Path.cwd() / ".env"` instead of `load_dotenv()` auto-discovery
- **Warning suppression**: Only warn for user-defined evaluator naming conflicts, not built-ins
- **Check command**: Accurate .env variable count display
- **Exception handling**: Improved error messages with UnicodeDecodeError handling
- **Test environment**: `run_cli` fixture handles both system pytest and venv pytest
- **Task matching**: Boundary-aware matching prevents ADV-1 from matching ADV-10

### Features
- **scripts/project**: Task lifecycle management CLI from agentive-starter-kit
- **Agent workflows**: Testing, commit, and review workflows
- **Agent templates**: Task starter and review starter templates
